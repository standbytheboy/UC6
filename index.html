UC6

Desenvolver algoritmos

Carga Horária: 96 horas

Indicadores

  1. Estrutura uma sequência lógica de ações de acordo com as etapas de resolução de um problema.
  2. Estrutura algoritmos, com base na lógica computacional e nos requisitos funcionais descritos no projeto.
  3. Seleciona e manipula estruturas de dados conforme as funcionalidades descritas no projeto de software.
  4. Cria sub-rotinas genéricas e reutilizáveis conforme as funcionalidades descritas no projeto de software.
  5. Valida algoritmo com base em valores de entradas e saídas de acordo com funcionalidades descritas do projeto de software.
  6. Seleciona a metodologia ágil de acordo com os requisitos do projeto de software

Aulas Práticas

  1. Módulo 1: Fundamentos e Ferramentas

    1. Introdução ao Ambiente de Desenvolvimento
    2. Conceitos Fundamentais de Algoritmos e Lógica de Programação
    3. Variáveis, Constantes e Tipos de Dados
    4. Estruturas Condicionais (if, else, switch-case)
    5. Estruturas de Repetição (while, do-while, for)

  2. Módulo 2: Estruturas de Dados e Manipulação Avançada

    1. Vetores e Listas
    2. Matrizes e Estruturas Multidimensionais
    3. Funções e Sub-rotinas
    4. Aplicação Prática: Algoritmos Clássicos com Estruturas de Dados
    5. Testes de Mesa e Depuração de Código

  3. Módulo 3: Estruturas Avançadas e Aplicações

    1. Manipulação Avançada de Arrays e Listas
    2. Estruturas de Dados Dinâmicas (Pilhas e Filas)
    3. Recursividade e Funções Recursivas
    4. Projeto 1: Simulação de Cadastro e Consulta de Dados
    5. Metodologias Ágeis: Scrum e Kanban
    6. Metodologias Ágeis na Prática

  4. Módulo 4: Orientação a Objetos

    1. Introdução à Programação Orientada a Objetos (POO)
    2. Encapsulamento, Herança e Polimorfismo
    3. Projeto 2: Sistema de Controle de Estoque
    4. Validação e Tratamento de Erros

  5. Módulo 5: Persistência de Dados com LocalStorage

    1. Armazenamento Local com LocalStorage
    2. CRUD com LocalStorage
    3. Projeto 3: Aplicação CRUD Completa
    4. Diagramação de Projetos com UML

  6. Módulo 6: Projetos Finais

    1. Planejamento do Projeto Final
    2. Desenvolvimento do Projeto Final - Parte 1
    3. Desenvolvimento do Projeto Final - Parte 2
    4. Apresentação e Feedback dos Projetos
